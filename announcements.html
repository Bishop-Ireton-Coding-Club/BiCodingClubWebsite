Coding Club

    Coding Club Takes on a Website! (8/13/2020)
  • Coding Club leaders have begun to code a website using HTML, CSS, and Javascript for anyone in any Computer Science related club or society. The website is meant to be a place for all Computer Science students at Bishop Ireton to come together and stay current on all clubs and CSHS. The website is still in the process of being coded, but it is expected to look good in the fall!

    • Projects Begin! (07/24/2020)
  • Coding Club has begun projects! Members and leaders have begun working on projects both in teams or individually. Projects are meant to be creative and can be anything at all using any language or software. Stay up to date on projects and Coding Club here.

    • Coding Club Weekly Meetings (6/5/2020)
  • During the summer, Coding Club has been having weekly meetings every Wednesday at 2pm. If you're interested, contact one of the leaders (Contacts page) and you'll be in! In coding club you will create projects, learn new programming languages, and be exposed to Computer Science topics.
